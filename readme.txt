Display an animated horizontal or vertical meter of donation goals with an easy to configure shortcode.

== Installation ==

Download the latest version of the Donation Goal Thermometer plugin and upload as a new plugin within your WordPress dashboard or via FTP to the wp-content/plugins folder and then activate the plugin.
